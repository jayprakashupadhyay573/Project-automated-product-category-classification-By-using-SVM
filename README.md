# Automated Product Category Classification using Machine Learning (SVM)

## 📌 Project Overview
This project is an end-to-end Machine Learning application that predicts the category of a product based on sales-related features.

The system uses a **Support Vector Machine (SVM)** model trained on a product sales dataset and provides predictions through a **Flask web interface**.

This project demonstrates the full ML workflow including:
- Data preprocessing
- Model training
- Model deployment using Flask
- GitHub project management
